# Bolt.new プロンプト - AuraTime デザイン生成

以下のプロンプトをBolt.newに貼り付けて使用してください。

---

## プロンプト本文

**AuraTime - マルチテナント型勤怠・給与管理システムのUIデザイン**

企業向けの勤怠・給与管理システム「AuraTime」のモダンでプロフェッショナルなWebアプリケーションUIを構築してください。

### 技術スタック
- **Next.js 16** (App Router, TypeScript)
- **React 19**
- **Tailwind CSS 4.x**
- **Shadcn UI** コンポーネントライブラリ
- **pnpm** パッケージマネージャー

### デザイン方針
- **モダンでプロフェッショナル**: 企業向けシステムとして信頼感のあるデザイン
- **使いやすさ重視**: 複雑な機能を直感的に操作できるUI
- **レスポンシブ対応**: デスクトップ・タブレット・モバイルに対応
- **アクセシビリティ**: WCAG 2.1 AA準拠
- **カラーパレット**: 落ち着いたブルー系をベース（信頼感）、アクセントカラーで重要操作を強調

### 主要画面構成

#### 1. ログイン画面
- シンプルで洗練されたログインフォーム
- メールアドレス・パスワード入力
- エラーメッセージ表示エリア
- ローディング状態の表示

#### 2. ダッシュボード（従業員向け）
- **レイアウト**: サイドバー + メインコンテンツエリア
- **ヘッダー**: ロゴ、現在日時、ユーザー名、ログアウトボタン
- **サイドバー**: 
  - 🏠 ダッシュボード
  - ⏱ 打刻・勤怠
  - 📅 休暇申請
  - 📄 給与明細
- **メインコンテンツ**:
  - クイック統計カード（今月の稼働時間、残業合計、有給残数）
  - 打刻エリア（現在時刻表示、出勤/退勤/休憩入/休憩戻ボタン）
  - 直近の打刻履歴一覧（10件程度）

#### 3. 打刻画面
- 大きな時計表示（現在時刻）
- 4つの打刻ボタン（出勤、退勤、休憩入、休憩戻）
- ボタンの状態管理（出勤済みなら退勤ボタンを有効化など）
- 打刻履歴のカレンダー表示
- 日別の労働時間サマリー

#### 4. 日次勤怠一覧画面
- カレンダービューまたはリストビュー
- 日別の労働時間、残業時間、休憩時間の表示
- 修正申請ボタン（承認済み・締め済みの場合は無効化）
- フィルタリング機能（期間指定）

#### 5. 休暇申請画面
- 休暇申請フォーム（休暇種別、開始日、終了日、理由）
- 休暇残数表示
- 申請履歴一覧
- カレンダー上での休暇期間の可視化

#### 6. 給与明細一覧画面
- 給与期間別の明細一覧
- 各明細のサマリー表示（支給額、控除額、差引支給額）
- 明細詳細へのリンク
- PDFダウンロードボタン

#### 7. 承認待ち一覧（マネージャー向け）
- 承認待ちの申請一覧
- 申請種別（勤怠修正、休暇申請）の表示
- 申請者、申請日、内容のサマリー
- 承認/却下ボタン
- フィルタリング（申請種別、申請者）

#### 8. シフト管理画面（マネージャー向け）
- カレンダービューでのシフト表示
- シフトテンプレートの作成・編集
- 従業員へのシフト割当
- ドラッグ&ドロップでのシフト調整

#### 9. 給与期間管理画面（管理者向け）
- 給与期間一覧
- 期間のステータス表示（Open, Closed, Calculated, Finalized）
- 締め操作ボタン
- 給与計算ジョブの実行ボタン
- ジョブ実行状況の表示

#### 10. マスタ管理画面（管理者向け）
- タブ切り替え（従業員、グループ、シフトテンプレート）
- 一覧表示（テーブル形式）
- 新規作成・編集・削除ボタン
- 検索・フィルタリング機能

#### 11. 監査ログ画面（システム管理者向け）
- ログ一覧（テーブル形式）
- フィルタリング（対象種別、操作種別、実行者、期間）
- ページネーション
- 詳細表示モーダル

### UIコンポーネント要件

#### 共通コンポーネント
- **ヘッダー**: 固定ヘッダー、ロゴ、ナビゲーション、ユーザーメニュー
- **サイドバー**: 折りたたみ可能、ロール別メニュー表示
- **カード**: 統計情報、サマリー表示用
- **テーブル**: ソート、ページネーション、フィルタリング対応
- **フォーム**: バリデーション、エラー表示、ローディング状態
- **モーダル**: 確認ダイアログ、詳細表示
- **トースト通知**: 成功/エラーメッセージ表示
- **カレンダー**: 日付選択、期間選択、イベント表示

#### 状態管理
- 認証状態（ログイン/ログアウト）
- ユーザー情報（ロール、会社ID）
- ローディング状態
- エラー状態

### レスポンシブデザイン
- **デスクトップ**: サイドバー常時表示、2カラム/3カラムレイアウト
- **タブレット**: サイドバー折りたたみ可能、1-2カラムレイアウト
- **モバイル**: ハンバーガーメニュー、1カラムレイアウト、タッチ操作最適化

### カラーパレット提案
- **プライマリ**: ブルー系（#2563eb, #1e40af）
- **セカンダリ**: グレー系（#6b7280, #374151）
- **成功**: グリーン（#10b981）
- **警告**: イエロー（#f59e0b）
- **エラー**: レッド（#ef4444）
- **背景**: ライトグレー（#f9fafb）、ホワイト（#ffffff）

### 実装の優先順位
1. **Phase 1（MVP）**: ログイン、ダッシュボード、打刻画面、日次勤怠一覧
2. **Phase 2**: 休暇申請、承認待ち一覧、シフト管理
3. **Phase 3**: 給与明細、給与期間管理、マスタ管理
4. **Phase 4**: 監査ログ、システム管理画面

### 注意事項
- マルチテナント対応のため、会社IDによるデータ分離を考慮
- ロールベースアクセス制御（RBAC）を実装
- セキュリティを重視（XSS、CSRF対策）
- パフォーマンス最適化（コード分割、遅延読み込み）

### 期待する成果物
- Next.jsプロジェクト構造
- 主要画面の実装（ログイン、ダッシュボード、打刻画面）
- Shadcn UIコンポーネントの統合
- レスポンシブデザインの実装
- 基本的なルーティングとナビゲーション

## 使用方法

1. 上記のプロンプト本文をコピー
2. https://bolt.new/ にアクセス
3. プロンプトを貼り付けて生成を開始
4. 生成されたコードを確認し、必要に応じて調整
